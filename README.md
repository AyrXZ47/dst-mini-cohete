# 🚀 DST Mini Cohete

[![License: CC BY 4.0](https://img.shields.io/badge/License-CC%20BY%204.0-lightgrey.svg)](LICENSE-MEDIA)

Mechanical design and aerodynamic simulation files for the Dragons Space Team's experimental mini rocket. 

## 📐 Overview
This repository contains the CAD models and flight simulations for the mini rocket platform. It is designed to be a testbed for avionics and the `dst-ignition-system`.

## 📂 Repository Structure
* `/hardware/`: Contains CAD models (`.FCStd`, `.obj`) built in FreeCAD.
* `/simulations/`: OpenRocket (`.ork`) flight simulations and CGNS aerodynamic flow files.
* `/docs/`: Renders and manufacturing instructions.

## 🛠️ Software Requirements
* **OpenRocket:** For editing and running the `.ork` flight simulations.
* **FreeCAD:** For modifying the 3D mechanical structures (`.FCStd`).
